# 2. 拉勾网招聘信息
1. 通过scrapy-redis框架编写爬虫,远程搭建redis单机/集群
2. 拉勾网的信息放在json中,通过post请求发送搜索的职位关键字和页码,获取json信息
3. 拉勾网反爬针对频繁操作,建议爬取间隔尽量长,更换请求头
